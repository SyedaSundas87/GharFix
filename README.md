# KhidmatGaar - خدمتگار

> AI-powered home services platform for Pakistan — built with React + Vite + TypeScript.

## 🚀 Quick Start

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Set the `GEMINI_API_KEY` in `.env` (copy from `.env.example`)
3. Run the development server:
   ```bash
   npm run dev
   ```

App runs at: `http://localhost:3000`

## 🛠️ Tech Stack
- React 19 + TypeScript
- Vite 6 + TailwindCSS v4
- Express.js (API proxy server)
- Motion (Framer Motion)
- Lucide Icons
- PWA-ready (manifest + service worker)

## 📱 Features
- AI-powered service intent detection (NLP)
- Multi-language support: English, Urdu, Roman Urdu
- Voice input (Web Speech API)
- Provider matching & booking
- Live service tracking
- Booking management (cancel, dispute, reschedule)
- Provider dashboard
- PWA installable

See `.env.example` for required variables.
